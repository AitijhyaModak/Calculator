
# Calculator

A website that implements a basic Calculator.


## Built Using

- HTML

- CSS

- Javascript


## Run this project

Create a project directory
```bash
  mkdir newProject
```

Clone the project

```bash
  git clone git@github.com:AitijhyaModak/Calculator.git
```

Go to the project directory

```bash
  cd newProject
```

Run the index.html file.
